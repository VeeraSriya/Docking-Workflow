# Molecular Docking of COX-2 with Celecoxib

## 🎯 Objective
To study the binding interaction between COX-2 (3LN1) protein and Celecoxib using molecular docking.

---

## 🧬 Protein Target
- Name: Cyclooxygenase-2 (COX-2)
- PDB ID: 3LN1
- Source: RCSB PDB

---

## 💊 Ligand
- Name: Celecoxib
- Type: COX-2 inhibitor (anti-inflammatory drug)

---

## 🛠 Tools Used
- AutoDock Vina 1.2.0
- Biopython
- Open Babel
- Python (Colab)

---

## ⚙ Methodology
1. Protein downloaded from RCSB PDB
2. Protein cleaned and converted to PDBQT
3. Ligand prepared and converted to PDBQT
4. Binding site defined using centroid method
5. Molecular docking performed using AutoDock Vina
6. Results analyzed and visualized

---

## 📊 Results
- Best Binding Energy: **-6.156 kcal/mol**
- Multiple stable binding poses generated
- Ligand successfully docked into COX-2 binding pocket

---

## 🧠 Conclusion
Celecoxib shows moderate binding affinity towards COX-2 receptor.  
The docking workflow successfully demonstrates structure-based drug design principles.
